# Todo Card — Stage 0

A single to-do card component built with plain HTML, CSS, and JavaScript. No frameworks.

## What's inside

- `index.html` — the card markup and edit modal
- `style.css` — mobile-first styles
- `todo.js` — all the interactivity

## How to run

Just open `index.html` in your browser. No build step, no installs.

## What it does

- Shows a task card with title, description, priority, status, due date, tags and a live time remaining counter
- Clicking **Edit** opens a modal where you can update any of the fields
- Clicking **Save** updates the card in place
- The checkbox marks the task as done and strikes through the title
- **Delete** removes the card after a confirmation

## Accessibility

- Keyboard navigable throughout
- Focus is trapped inside the modal when it's open
- Screen reader friendly labels on all interactive elements
